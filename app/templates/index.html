RSA-2048 + PSS Signatures

AES-256-GCM Encryption
Authenticated encryption (AEAD). 128-bit GCM tag detects any tampering. NIST SP 800-38D.

ℹ️ How It Works
1️⃣
Generate session key
os.urandom(32) → 256-bit AES key. Ephemeral, used once.
2️⃣
AES-256-GCM encryption
Encrypts document. GCM tag (128-bit) ensures integrity.
3️⃣
RSA-OAEP key wrap
Session key encrypted with recipient's RSA-2048 public key.
4️⃣
Output .enc bundle
[4B key_len][enc_key][12B IV][ciphertext+GCM_tag]
