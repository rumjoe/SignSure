# Contributing to SignSure

Thank you for your interest! All contributions are welcome.

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes with proper docstrings and comments
4. Run the test suite: `pytest tests/ -v`
5. Submit a pull request

## Code Standards
- Python 3.10+ compatible
- PEP 8 style (use `black` for formatting)
- All new features need corresponding tests in `tests/`
- Cryptographic changes must cite the relevant standard (NIST/RFC)

## Security Issues
Report security issues privately via email — do not create public issues.
